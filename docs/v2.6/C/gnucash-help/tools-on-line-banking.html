8.5. HBCI (Online Banking) Setup Assistant

Note

This section is "under construction - any input will be welcome !!".

The HBCI Assistant is used to create and edit data to enable access to Online banking transactions. Currently the best instruction for this process are in the GnuCash wiki at Setting up OFXDirectConnect in GnuCash 2. If HBCI Setup does not appear in your Tools menu, verify that you are in an Account Tree or Register tab.
