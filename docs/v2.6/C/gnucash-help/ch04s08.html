4.8. Business Windows

4.8.1. Customers Overview

To open the Customers Overview window, click Business → Customer → Customers Overview.

Table 4.35. Customers Overview - Toolbar Buttons

Toolbar Buttons

Description

New

Create a new customer

Edit

Edit the selected customer

New Invoice

Create a new invoice (by default it is for the selected customer)

Customer Listing

Show customer aging overview for all customers


Table 4.36. Customer context menu

Context menu item

Description

Rename Page

Edit Customer

Edit the selected customer

New Invoice

Create a new invoice

Customer Report

Show customer report


Note

The customer context menu appears if you right click on a customer in the list.
